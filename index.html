Sistema de Monitoreo en Tiempo Real para Máquinas Tragamonedas

Este proyecto tiene como objetivo prevenir fraudes en las máquinas tragamonedas de casino mediante el uso de tecnología. A través de un microcontrolador ESP32, el sistema monitorea en tiempo real las entradas y salidas de los contadores de las máquinas, detectando cualquier irregularidad o intento de manipulación fraudulenta, como el uso de dispositivos conocidos como 'hammer'.

La información recopilada se envía automáticamente a una página web, donde los operarios del casino pueden visualizar los datos en tiempo real, permitiéndoles tomar acciones inmediatas si se detecta algún fraude. Este sistema mejora la seguridad de las máquinas tragamonedas, protegiendo los ingresos del casino y garantizando la equidad en el juego. Además, su diseño escalable permitirá la integración de múltiples máquinas, asegurando un control eficiente y automatizado.
